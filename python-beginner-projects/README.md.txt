# Python Beginner Projects

This repository contains beginner Python projects I built while learning Python.

## Projects Included

1. **Number Guessing Game**  
   Guess a random number between 1 and 10.

2. **Calculator**  
   A simple calculator that can add, subtract, multiply, and divide.

3. **To-Do List**  
   A console program to add, remove, and view tasks.

4. **Currency Converter**  
   Converts USD, EUR, and GBP based on predefined exchange rates.

5. **Personal Info Display**  
   Displays personal information (name, age, city) entered by the user.

## How to Run
- Open the project file in Python and run it.
